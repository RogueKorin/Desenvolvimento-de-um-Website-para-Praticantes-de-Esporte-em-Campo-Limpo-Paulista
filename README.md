# Desenvolvimento-de-um-Website-para-Praticantes-de-Esporte-em-Campo-Limpo-Paulista

## Descrição
Nosso trabalho busca auxiliar comunidades esportivas em Campo Limpo Paulista a se encontrarem, organizarem e praticarem esportes, auxiliando com horários, eventos, agendas e fórums/chats.

## Como executar o projeto (passo-a-passo)

Instalar as dependencias e aplicativos

1 - Baixe o visual studio code nesse link: https://code.visualstudio.com/Download

2 - Baixe o MongoDB Compass Server por esse link: https://www.mongodb.com/try/download/community

3 - Baixe o Nodejs por esse link: https://nodejs.org/en/download

4 - Baixe o terminal gitbash no seguinte link: https://git-scm.com/install/

5 - Após a instalação do software necessário, rode o visual studio code

6 - Abra o terminal gitbash, na pasta backend e instale as dependências: npm install socket.io multer mongoose jsonwebtoken express dotenv cors bcrypt

7 - Abra outro terminal separado na pasta frontend e instale suas dependências: npm install axios fullcalendar bootstrap

8 - Para rodar o backend, ainda no seu terminal aberto, use o comando "node app"

9 - Para rodar o frontend, selecione seu terminal e use o comando "npm run dev". Após isso, um link será exibido para o localhost do site, que deve te levar ao website do projeto rodando.
